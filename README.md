## HackerNews Viewer
Browse HN stories with the arrow keys (<- and ->), open the link pressing space bar o enter.

Check how it looks [here](http://hacker-news-viewer.miguel.im)

###Development
```
$ npm start
```

Start hacking away!

###Building
To build `bundle.js` in production (required for react):

```
$ webpack -p
```
